## 设置页设计

设置页定位为“模型与录音控制台”，按业务职责分区：

1. 本地 ASR
   - runtime 探测结果：`whisperkit-cli`、`argmax-cli`。
   - 首选 runtime：优先可用的 `whisperkit-cli`，其次 `argmax-cli`。
   - 模型选择：默认 `large-v3-v20240930_626MB`，允许选择小模型。
   - 下载/重试入口：委托 CLI 执行，应用记录状态。
   - 缓存状态：目录、更新时间、可用性和错误摘要。

2. 录音设置
   - 录音开关。
   - 切片时长。
   - 无效录音触发秒数。
   - 语言。

3. MiniMax M3
   - 语义 base URL。
   - 模型名。
   - API Key 脱敏状态。
   - 语义测试连接。

4. 隐私边界
   - 本地音频：音频默认留在本机。
   - 本地转写：runtime 和模型缓存均在本机。
   - 云端语义：MiniMax M3 只接收转写后的文本上下文。

设置页不再展示火山 ASR 下拉和云端兜底开关。相关旧字段可留在后端用于兼容和迁移，但默认 UI 不暴露。

## 本地 ASR Runtime 与模型状态

### Runtime 探测

后端新增或扩展命令用于探测：

1. `whisperkit-cli`
2. `argmax-cli`

探测结果至少包括：

```text
runtime_id
available
path
version
error_message
```

如果两个 runtime 都缺失，设置页提示安装 `brew install whisperkit-cli` 或配置 `argmax-cli` 路径。

### 模型下载

下载策略：

1. 应用内保存目标模型、下载状态、缓存目录、错误摘要。
2. 实际下载/初始化优先通过 CLI 完成。
3. 应用不直接实现 Hugging Face 大文件下载作为默认路径。
4. 下载失败后支持重试。

模型状态：

```text
not_started
downloading
available
failed
```

默认模型：

```text
large-v3-v20240930_626MB
```

允许小模型选项：

```text
base
tiny
```

具体 CLI 支持的参数以本机 runtime 探测结果和命令返回为准；当 CLI 不支持目标模型时，显示明确错误并建议切换模型。

### 本地转写

如果 runtime 与模型均可用：

1. 录音片段可发起本地 ASR 转写。
2. 转写结果写入现有 `transcript_segments` / `transcript_jobs`。
3. 失败时保留错误摘要，并允许从录音片段页重试。

如果 runtime 或模型不可用：

1. 不显示“本地模型可用”的假状态。
2. 不静默走云端 ASR。
3. 明确提示缺失项和下一步。

## 录音片段页最小功能

录音片段页替代旧“转写评估”入口。

核心功能：

1. 展示录音片段列表。
2. 展示每个片段的转写任务状态、模型名、错误摘要。
3. 选中片段后展示转写文本、置信度和 speaker label。
4. 失败任务可重试。
5. 展示说话人列表和基础标签。
6. 空态文案改为“开始录音后将在这里看到片段和转写状态”。

不做：

1. 不展示用户可见本地音频导入栏。
2. 不做说话人合并。
3. 不做声纹训练。
4. 不做跨会话别名管理。

## 错误处理

| 错误 | 用户提示 | 行为 |
| --- | --- | --- |
| `runtime_not_found` | 未检测到 `whisperkit-cli` / `argmax-cli` | 提供安装或配置路径提示 |
| `model_not_downloaded` | 本地 ASR 模型未下载 | 显示下载按钮 |
| `download_failed` | 模型下载失败 | 显示错误摘要和重试 |
| `model_available_runtime_missing` | 模型已缓存，但 runtime 缺失 | 阻止转写并提示修复 |
| `transcription_failed` | 本地转写失败 | 片段任务标记失败，显示重试 |
| 浏览器原型 | 浏览器原型无法执行本地 runtime 探测，请在桌面端验证 | 不再提示云模型测试 |

错误摘要不得包含完整音频路径、API Key 或完整隐私文本。

## 风险与规避

| 风险 | 影响 | 规避 |
| --- | --- | --- |
| CLI 参数在不同版本间不一致 | 下载或转写命令失败 | 先探测版本和 help 输出；错误态可见 |
| 本机未安装 runtime | 无法真实转写 | 缺失态明确，不伪造成功 |
| 模型下载耗时较长 | 用户误以为卡住 | 显示阶段状态、更新时间和失败重试 |
| 旧 ASR 云端字段仍存在 | 用户误解云端仍是主路径 | UI 移除火山 ASR 用户入口，文案明确本地主线 |
| 录音片段页范围膨胀 | v1.2.1 难以收口 | 只做查看、状态和基础重试 |
