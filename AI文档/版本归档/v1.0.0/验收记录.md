# v1.0.0 验收记录

## 结论

v1.0.0 产品闭环版本已通过当前最小验证、code review 和功能 review。导出中心可生成 Markdown、SRT、JSON 和本地分享快照；会话归档支持搜索；设置页展示 provider 成本、隐私说明和密钥状态。

## In Scope

1. Markdown / SRT / JSON 导出。
2. 本地分享快照。
3. `external_exports` 导出记录。
4. 会话归档与搜索。
5. 错误恢复和任务重试 UI 完整化。
6. 设置页 provider 成本、隐私说明、密钥状态。
7. Apple 风格视觉参考图。

## Out of Scope

1. 真实系统文件保存对话框。
2. 云分享、外链发布、外部同步。
3. 多语言导出。
4. v1.1 多会话知识库和复盘周报。
5. 播客 / TTS。

## Code Review

未发现阻塞问题。

已检查项：

1. 版本目标与迭代计划一致，只实现 v1.0。
2. command / app / domain / provider 分层没有倒置。
3. 导出 DTO 与前端类型字段一致。
4. 导出服务不记录 API Key，不暴露完整本地音频路径。
5. 分享快照只生成本地 HTML，不接入云分享。
6. UI 覆盖导出生成、格式切换、快照、来源、状态、重试入口、归档搜索和设置状态。
7. 未修改生产配置、证书、账号或外部发布流程。
8. 未重新引入旧 Qwen / llama.cpp Todo runtime。

已处理项：

1. 补齐 `domain/export.rs`、`app/export_service.rs`、`commands/export.rs` 和 `providers/export/local_file.rs`。
2. 新增 `external_exports` 表，满足本地导出记录要求。
3. 新增导出中心与会话归档搜索。
4. 新增设置页 provider 成本、隐私和密钥状态。

剩余风险：

1. 文件保存仍是内容预览，未接系统文件保存对话框。
2. 快照没有外链权限、撤回、访问控制。
3. SRT 长音频字幕边界还需要真实样本扩展验证。
