# v0.8.0 验收记录

## 结论

v0.8.0 思维脑图版本已通过最小验证、code review 和功能 review。脑图可生成、编辑、折叠、追溯来源，并可导出 Markdown / JSON。用户编辑会生成新 artifact 版本，不覆盖原始生成结果。

## In Scope

1. `MindMapView` / “思维脑图”页面。
2. `semantic_artifacts(type='mind_map')` 写入和读取。
3. Mind map JSON 结构：root、nodes、edges、summary、source_spans。
4. 节点编辑、折叠、重新生成。
5. Markdown / JSON 导出。
6. Apple 风格视觉参考图。

## Out of Scope

1. 真实在线 MiniMax M3 脑图生成调用。
2. 独立 `mind_maps` 表。
3. Moment、Deep Research。
4. 分享快照、跨设备同步、多语言导出。

## Code Review

未发现阻塞问题。

已检查项：

1. 版本目标与迭代计划一致，只实现 v0.8。
2. command / app / domain 分层没有倒置。
3. `semantic_artifacts` 与前端类型字段一致。
4. 用户编辑不会被重新生成静默覆盖。
5. Markdown / JSON 导出基于 artifact payload。
6. UI 覆盖生成、编辑、折叠、来源、导出状态。
7. 未新增敏感日志或生产配置改动。

已处理项：

1. 边关系条最初位于画布 grid 内，截图中视觉像被裁切；已移动到画布上方并重新验证。
2. 最新脑图读取逻辑最初会吞掉 SQLite 查询错误；已改为显式错误返回。
3. 节点标题最初只在前端做空值拦截；已增加后端空标题校验。

剩余风险：

1. 真实 MiniMax M3 mind map 结构化调用尚未接入。
2. 来源片段按钮当前只跳转到转写评估页，未自动滚动到具体片段。
3. 复杂多会话合并策略当前使用统一 artifact 结构承载，后续仍需真实 provider 输入策略验证。
