1. `src-tauri/src/commands/`：Tauri command 边界。
2. `src-tauri/src/app/`：设置服务与查询服务。
3. `src-tauri/src/domain/`：前端 DTO、provider 描述、运行状态、会话、Todo、转写记录等领域契约。
4. `src-tauri/src/infra/sqlite.rs`：SQLite 初始化、迁移、`semantic_artifacts`、`model_invocations`。
5. `src-tauri/src/providers/`：本地 ASR、Speaker、MiniMax M3 语义 provider 描述。
6. `src-tauri/src/jobs/todo_extraction.rs`：Todo 语义产物登记边界。
7. `src/App.tsx` 与 `src/styles.css`：设置页 provider 与隐私边界展示。
