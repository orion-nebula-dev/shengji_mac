# v1.1.1 验收记录

## 结论

v1.1.1 完成应用品牌和安装包修复：应用名为“声记”，图标已替换，安装包重新生成并完成本地校验。

## 验收项

1. 应用包名显示为“声记”。
2. 窗口标题显示为“声记”。
3. `Info.plist` 版本为 `1.1.1`。
4. 新图标资源写入 `src-tauri/icons/`。
5. 本地安装包输出到 `release/声记_1.1.1_aarch64.app.zip`。
6. 从 zip 解压出的 `声记.app` 通过本地启动 smoke test。

## 验证命令

```bash
npm run build
cargo check --manifest-path src-tauri/Cargo.toml
npm run tauri:build
codesign --verify --deep --strict --verbose=2 <unzipped app>
```

## 已知限制

1. 本地安装包未做 Developer ID 签名和公证。
2. 安装包不提交 Git；正式分发时应作为 Release 附件上传。
3. 本次推荐使用 `.app.zip`；Tauri fancy DMG 不作为推荐交付物。
