# v1.1.0 验收记录

## 结论

v1.1.0 已完成翻译与多语言导出版本范围：转写翻译、摘要翻译、多语言导出模板、来源追溯、视觉参考、code review、功能 review 和最小验证。

## 范围

In Scope：

1. `translation` 语义产物。
2. 转写翻译与摘要翻译。
3. 多语言 Markdown / JSON / Snapshot 导出模板。
4. 来源追溯展示。
5. Export Center v1.1 交互与视觉优化。

Out of Scope：

1. 播客脚本。
2. TTS provider。
3. 音频生成入口。
4. 云端分享和外部同步。

## 验收路径

1. 打开“导出中心”。
2. 选择 `English / en-US`。
3. 点击“生成翻译”。
4. 点击“多语言导出”。
5. 查看多语言 Markdown / JSON / Snapshot 预览。
6. 查看右侧“翻译来源追溯”。
7. 打开“设置”，确认没有 TTS provider。

## Code Review 处理

1. 确认摘要翻译不会覆盖原始摘要。
2. 确认翻译结果通过 `sourceSegmentId`、说话人和时间范围追溯原始片段。
3. 确认多语言导出只在存在目标语言翻译产物时生成。
4. 确认 v1.1 没有引入播客、TTS 或音频生成入口。
5. 确认快照不暴露完整本地路径、API Key 或外部账号信息。

## 剩余风险

1. 翻译内容当前为 deterministic 本地语义模拟。
2. 真实 MiniMax M3 翻译质量和 schema 强校验需后续接入。
3. 多语言导出仍是模板预览，不包含系统保存面板。
