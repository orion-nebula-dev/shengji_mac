# AI文档索引

本目录存放声记项目的产品、技术、版本、发布和规范文档。文档按用途分区，旧方案和过时资料进入 `废纸篓/`，发布版本资料进入 `版本归档/`。

## 目录说明

| 目录 | 用途 |
| --- | --- |
| `00-项目总览/` | 项目说明、工作区目录规范、全局索引 |
| `01-产品方案/` | 产品定位、用户流程、PRD |
| `02-技术方案/` | 架构方案、数据模型、Provider 设计、外部能力评估依据 |
| `03-版本迭代/` | 版本路线、版本目标、代码归档方案 |
| `04-发布记录/` | 各版本发布说明 |
| `05-规范制度/` | 开发规范、Git 规范、协作规则 |
| `06-验证报告/` | 能力验证、实验记录、评估报告 |
| `设计参考/` | 前端视觉参考图、风格说明、设计资产来源 |
| `版本归档/` | 每个正式版本的归档清单、验收记录、已知问题 |
| `废纸篓/` | 已废弃但需要保留历史依据的旧方案 |

## 维护规则

1. 新产品需求放入 `01-产品方案/`。
2. 新技术设计放入 `02-技术方案/`。
3. 新版本路线和目标放入 `03-版本迭代/`。
4. 发布说明统一放入 `04-发布记录/`。
5. 验证报告和实验记录放入 `06-验证报告/`。
6. 被替代的旧文档移动到 `废纸篓/YYYY-MM-DD-说明/`。
