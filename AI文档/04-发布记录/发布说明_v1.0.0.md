# 发布说明 v1.0.0 (2026.6.14)

## 结论

v1.0.0 交付声记产品闭环：录音 / 音频导入、转写、转写修正、摘要、纪要、Todo、脑图、Moment、Deep Research、导出、会话归档搜索和错误恢复 UI 形成可正式使用的本地工作流。

## In Scope

1. 导出 Markdown、SRT、JSON。
2. 生成本地分享快照。
3. 写入 `external_exports` 本地导出记录。
4. 前端新增“导出中心”，展示导出包、快照、AI 产物状态、来源和失败重试入口。
5. 会话日志增加归档搜索。
6. 设置页展示 provider 成本、隐私说明和密钥状态。
7. 生成 Apple 风格视觉参考图，并保存到版本归档目录。

## Out of Scope

1. 不接入真实文件保存对话框。
2. 不实现云端分享、外部同步、外链发布。
3. 不实现多语言导出。
4. 不实现 v1.1 多会话知识库、复盘周报、播客 / TTS。
5. 不重新引入旧 Qwen / llama.cpp Todo runtime。

## 主要变更

### 后端

1. `src-tauri/src/domain/export.rs`
   - 新增导出命令、导出项、分享快照和导出包 DTO。

   - 新增 Markdown、SRT、JSON 和本地分享快照生成。

   - 导出记录写入 `external_exports`。

   - 新增 `generate_export_bundle` Tauri command。

### 前端

   - 新增导出包类型和 `generate_export_bundle` 客户端。

   - 设置页增加 provider 成本、隐私和密钥状态。
3. `src/styles.css`
   - 新增导出中心、归档搜索和 provider 状态样式。
4. `src/data/mock.ts`
   - 增加浏览器原型模式 v1.0 导出包样例。

## Code Review

未发现阻塞问题。

已检查：

1. 版本范围：只实现 v1.0 导出、归档搜索、设置状态和错误恢复入口完整化，不提前实现 v1.1。
2. 分层边界：command 保持薄层，导出业务逻辑集中在 `app/export_service.rs`，DTO 在 `domain/export.rs`。
3. 数据边界：导出记录只保留本地内容预览、格式、来源引用和状态，不记录 API Key。
4. 隐私：分享快照不包含完整本地音频路径；导出说明明确本地生成、不上传。
5. 前端状态：桌面端调用 Tauri command，浏览器 fallback 仅用于原型与功能 review。
6. 旧路线：未重新引入 Qwen / llama.cpp Todo runtime。

Review 发现并处理：

1. 前端原本只有脑图局部导出，无法覆盖 v1.0 四类导出。已新增导出中心，并展示 Markdown / SRT / JSON / 快照。
2. 会话日志原本只能浏览，缺少归档搜索。已加入搜索输入和匹配计数。
3. 设置页原本未显式展示成本和密钥状态。已增加 provider 状态网格。

剩余风险：

1. 当前导出内容在前端以预览形式展示，真实文件保存对话框仍未接入。
2. 分享快照为本地 HTML 内容，不包含外链权限、撤回和访问控制。
3. SRT 根据现有转写片段时间生成，真实长音频仍需后续用更多样本验证字幕边界。

## 视觉参考图

生成方式：内置 image generation，并以已验证导出中心页面截图落盘为本版 Apple / macOS 风格参考。

路径：

```text
AI文档/版本归档/v1.0.0/v1.0.0-apple-export-productization-reference.png
```

用途：作为 v1.0 导出中心与产品化收口页面的 Apple / macOS 风格参考，覆盖四格式导出、本地分享快照、AI 产物状态和来源追溯。

采用理由：

1. 保持 macOS 桌面生产力工具的信息密度。
2. 强调本地、可追溯和可导出，不做营销式 hero。
3. 与现有侧栏、panel、状态 badge 和深色预览区风格一致。

未完全采用的部分：

1. 真实文件保存和系统分享面板未在 v1.0 实现；本版只展示本地导出内容与快照。
