# 发布说明 v0.3.0

> 项目：声记 Mac App / 智能 Todo
> 版本：v0.3.0
> 日期：2026-04-25

## 1. 发布目标

v0.3.0 聚焦两件事：

1. 将 `.github` 下的 agent、instruction、CI 配置改为符合当前声记项目的 Codex 协作规范。
2. 在不重写底层数据流的前提下，按“方案 A”优化当前可运行界面，使其更接近声记 v2.0 的 macOS 效率工具设计基线。

## 2. 主要变更

### 2.1 Codex 与项目规范

1. 新增 `.github/AGENTS.md` 作为 Codex 协作入口。
2. 重写 `.github/copilot-instructions.md`，移除 PromoFlow、飞书、FastAPI、PostgreSQL、OSS 等不相关内容。
3. 重写 `.github/agents/feat.agent.md`、`fix.agent.md`、`review.agent.md`，适配声记当前 Tauri + React + Rust 项目。
4. 重写 `.github/instructions/frontend/*`，指向当前 `src/` 前端结构。
5. 重写 `.github/instructions/backend/*`，指向 `src-tauri/src`、Tauri command、本地数据、日志脱敏和模型运行时。
6. CI 改为根目录前端构建与 `src-tauri` Rust 检查。

### 2.2 当前界面优化

1. 将原三 tab 原型改为 macOS 风格窗口结构。
2. 新增左侧固定导航：`今日工作台`、`行动中心`、`会话日志`、`系统状态`、`设置`。
3. `今日工作台` 展示录音状态、Todo 数量、最近切片、最近提取和上一段会话。
4. `行动中心` 保留 Todo 搜索、筛选、详情和状态切换。
5. `会话日志` 保留会话文稿、触发原因、Provider、回退状态和回退原因。
6. `系统状态` 汇总桌面桥接、本地模型运行时、失败与回退记录。
7. `设置` 保留录音配置、ASR 配置、Todo 提取配置、本地模型版本和云端兜底开关。

### 2.3 版本号

1. `package.json` 更新为 `0.3.0`。
2. `package-lock.json` 根包版本更新为 `0.3.0`。
3. `src-tauri/tauri.conf.json` 更新为 `0.3.0`。
4. `src-tauri/Cargo.toml` 更新为 `0.3.0`。

### 2.4 本地优先模型策略

1. ASR 默认改为 `local` 本地优先。
2. Todo 默认改为 `embedded_local` 本地优先。
3. `allow_cloud_fallback` 作为 ASR 与 Todo 的统一云端兜底开关。
4. 开启兜底时，本地不可用后可回退云端。
5. 关闭兜底时，ASR 与 Todo 都保持纯本地，不会静默调用云端。
6. 当前本地 ASR 尚未正式接入，纯本地 ASR 会明确失败并提示。

## 3. 保留能力

本版本明确保留日志与排障相关能力：

1. 会话文稿。
2. Todo 来源文稿。
3. 提取路径。
4. 云端兜底状态。
5. 回退原因。
6. 失败状态。
7. 桌面桥接状态。
8. 本地 Todo 运行时状态。
