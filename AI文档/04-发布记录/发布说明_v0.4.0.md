# v0.4.0 (2026.6.14)

## 版本主题

架构止血与本地 ASR 路线切换。

## 主要变更

1. 后端建立 `commands` / `app` / `domain` / `infra` / `providers` / `jobs` 分层骨架。
2. 新增 ASR、Speaker、Semantic、Embedding、Export provider 边界。
3. Todo 入口固定到 MiniMax M3 语义边界，候选产物进入 `semantic_artifacts(type='todo_extraction')`。
4. 新增 `semantic_artifacts` 与 `model_invocations` 表，并补齐旧约束迁移。
5. 删除旧 Qwen / llama.cpp Todo runtime 运行入口和旧 Todo provider 兼容路径。
6. 设置页展示本地 ASR、本地说话人分离、云端 MiniMax M3 与预留能力的隐私边界。
7. 抽出 `app::query_service`，让启动、任务处理和录音 command 不再依赖根模块查询 helper。
