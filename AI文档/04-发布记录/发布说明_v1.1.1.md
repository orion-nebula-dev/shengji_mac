# 发布说明 v1.1.1 (2026.6.14)

## 结论

v1.1.1 是 v1.1.0 的安装包与品牌修复版本：应用名统一为“声记”，替换 macOS 应用图标，并重新生成可打开的本地安装包。

## 修复内容

1. Tauri `productName` 和窗口标题从“智能 Todo”改为“声记”。
2. bundle identifier 改为 `com.shengji.desktop`。
3. 默认本地数据目录改为 `~/Library/Application Support/com.shengji.desktop/`。
4. 使用 image generation 生成新的声波 + 文稿风格 app icon，并导出 Tauri 所需 PNG / ICNS 资源。
5. 版本号更新为 `1.1.1`。
6. `release/` 加入 `.gitignore`，安装包只作为本地 / Release 附件资产，不进入源码仓库。

## 验证

```bash
npm run build
cargo check --manifest-path src-tauri/Cargo.toml
npm run tauri:build
codesign --verify --deep --strict --verbose=2 <unzipped app>
```

## 安装包说明

本次可用安装包：

```text
release/声记_1.1.1_aarch64.app.zip
release/checksums.sha256
```

已验证：从 zip 解压出的 `声记.app` 版本为 `1.1.1`，ad-hoc 签名一致，并通过本地启动 smoke test。Tauri 生成的 fancy DMG 不作为本次推荐交付物。

## 剩余风险

当前安装包为本机 ad-hoc 签名，未使用 Apple Developer ID 签名，也未做 Apple notarization。首次打开时 macOS 仍可能提示来自未识别开发者。
